## 是否需要购买域名

### ✅ 购买域名的优势
- **更专业**：`www.toolify.ai` 比 `toolify-website-7upf.vercel.app` 更专业
- **更易记**：用户更容易记住和分享
- **品牌形象**：提升网站可信度和品牌价值
- **SEO 友好**：有利于搜索引擎优化
- **长期使用**：不受 Vercel 免费域名限制

### ❌ 不购买域名的劣势
- 使用 Vercel 自动生成的域名（如 `toolify-website-7upf.vercel.app`）
- 域名不够专业
- 不利于品牌推广

**结论：如果网站要长期运营，建议购买域名。如果只是测试，可以不买。**

## 如何选择域名

### 1. 域名命名原则

**✅ 好的域名特点：**
- 简短易记（2-3 个词）
- 与品牌/业务相关
- 容易拼写
- 没有连字符（除非必要）

**❌ 避免的域名：**
- 太长（超过 15 个字符）
- 太多连字符（如 `tool-ify-ai.com`）
- 数字和字母混用（如 `toolify123.com`）
- 容易拼错的词

### 2. 推荐域名示例

基于你的项目 "Toolify"，可以考虑：

**首选：**
- `toolify.ai` - 最简洁，与品牌一致
- `toolify.com` - 经典 .com 域名

**备选：**
- `aitoolify.com`
- `toolifyhub.com`
- `bestaitools.com`
- `toolify.io`

### 3. 域名后缀选择

| 后缀 | 特点 | 价格 | 推荐度 |
|------|------|------|--------|
| `.com` | 最常用，最权威 | $10-15/年 | ⭐⭐⭐⭐⭐ |
| `.ai` | AI 相关，很流行 | $30-70/年 | ⭐⭐⭐⭐ |
| `.io` | 科技公司常用 | $30-50/年 | ⭐⭐⭐⭐ |
| `.app` | 应用类网站 | $15-25/年 | ⭐⭐⭐ |
| `.net` | 备选方案 | $10-15/年 | ⭐⭐⭐ |
| `.org` | 非营利组织 | $10-15/年 | ⭐⭐ |

**建议：优先选择 `.com` 或 `.ai`**

## 购买域名详细步骤

### 方法一：Namecheap 购买（推荐）

#### 步骤 1：访问 Namecheap
1. 打开浏览器，访问：https://www.namecheap.com
2. 点击右上角 "Sign Up" 注册账号
3. 填写邮箱、密码等信息
4. 验证邮箱完成注册

#### 步骤 2：搜索域名
1. 在首页搜索框输入想要的域名（如：`toolify`）
2. 点击 "Search" 按钮
3. 查看可用域名和价格

#### 步骤 3：选择域名
1. 查看搜索结果：
   - ✅ 绿色 = 可用
   - ❌ 红色 = 已被注册
2. 选择你想要的域名和后缀
3. 点击 "Add to Cart"（加入购物车）

#### 步骤 4：配置域名
1. 在购物车页面，选择购买年限（通常 1 年）
2. **重要：勾选 "WhoisGuard"（免费隐私保护）**
3. 点击 "Confirm Order"

#### 步骤 5：付款
1. 选择支付方式：
   - 信用卡/借记卡
   - PayPal
   - 支付宝/微信支付（如果支持）
2. 填写支付信息
3. 完成付款

#### 步骤 6：验证邮箱
1. 检查邮箱，收到 Namecheap 的确认邮件
2. 点击邮件中的验证链接
3. 完成域名注册

### 方法二：阿里云购买（国内用户）

#### 步骤 1：注册账号
1. 访问：https://wanwang.aliyun.com
2. 注册/登录阿里云账号
3. 完成实名认证（必需）

#### 步骤 2：搜索和购买
1. 在搜索框输入域名
2. 查看可用域名
3. 选择域名，加入购物车
4. 完成支付

## 配置域名到 Vercel

购买域名后，需要将域名指向 Vercel。

### 步骤 1：在 Vercel 中添加域名

1. 登录 Vercel：https://vercel.com
2. 进入你的项目（toolify-website）
3. 点击 "Settings" 标签
4. 点击左侧菜单的 "Domains"
5. 在输入框中输入你的域名（如：`www.toolify.ai` 或 `toolify.ai`）
6. 点击 "Add" 按钮

### 步骤 2：查看 DNS 配置要求

Vercel 会显示需要配置的 DNS 记录，通常有两种情况：

**情况 A：使用 CNAME 记录（推荐，用于子域名如 www）**
```
类型：CNAME
名称：www
值：cname.vercel-dns.com
TTL：3600（或自动）
```

**情况 B：使用 A 记录（用于根域名）**
```
类型：A
名称：@（或留空）
值：76.76.21.21（Vercel 提供的 IP）
TTL：3600（或自动）
```

### 步骤 3：在域名服务商配置 DNS

#### 如果使用 Namecheap：

1. 登录 Namecheap：https://www.namecheap.com
2. 进入 "Domain List"
3. 找到你的域名，点击 "Manage"
4. 点击 "Advanced DNS" 标签
5. 添加 DNS 记录：

   **对于 www 子域名：**
   - 点击 "Add New Record"
   - Type: `CNAME Record`
   - Host: `www`
   - Value: `cname.vercel-dns.com`
   - TTL: `Automatic`
   - 点击保存

   **对于根域名（toolify.ai）：**
   - 点击 "Add New Record"
   - Type: `A Record`
   - Host: `@`
   - Value: `76.76.21.21`（Vercel 提供的 IP）
   - TTL: `Automatic`
   - 点击保存

#### 如果使用阿里云：

1. 登录阿里云控制台
2. 进入 "域名" → "域名解析"
3. 找到你的域名，点击 "解析设置"
4. 添加记录：

   **对于 www 子域名：**
   - 记录类型：`CNAME`
   - 主机记录：`www`
   - 记录值：`cname.vercel-dns.com`
   - TTL：`10 分钟`

   **对于根域名：**
   - 记录类型：`A`
   - 主机记录：`@`
   - 记录值：`76.76.21.21`
   - TTL：`10 分钟`

### 步骤 4：等待 DNS 生效

- DNS 生效通常需要 **几分钟到几小时**
- 可以在 Vercel 的 Domains 页面查看状态
- 显示 "Valid Configuration" 表示配置成功

### 步骤 5：测试访问

DNS 生效后：
1. 在浏览器访问你的域名（如：`https://www.toolify.ai`）
2. 应该能看到你的网站
3. Vercel 会自动配置 SSL 证书（HTTPS）

## 常见问题

### Q1: 域名价格是多少？

**A:** 价格因后缀而异：
- `.com`: $10-15/年（约 ¥70-100/年）
- `.ai`: $30-70/年（约 ¥200-500/年）
- `.io`: $30-50/年（约 ¥200-350/年）

**注意：** 首年价格可能较低，续费价格可能更高。

### Q2: 需要购买 SSL 证书吗？

**A:** 不需要！Vercel 免费提供 SSL 证书（HTTPS），自动配置。

### Q3: 可以只买域名，不买主机吗？

**A:** 可以！域名和主机是分开的。你只需要：
1. 购买域名
2. 配置 DNS 指向 Vercel
3. Vercel 免费提供主机服务

### Q4: DNS 配置后多久生效？

**A:** 通常：
- 最快：5-10 分钟
- 一般：30 分钟 - 2 小时
- 最长：24-48 小时（很少见）

### Q5: 可以同时使用 www 和根域名吗？

**A:** 可以！在 Vercel 中添加两个域名：
- `www.toolify.ai`
- `toolify.ai`

Vercel 会自动将根域名重定向到 www（或反之）。

### Q6: 域名到期了怎么办？

**A:** 
- 域名到期前会收到续费提醒
- 及时续费即可
- 过期后可能有 30 天的宽限期
- 超过宽限期域名可能被释放

### Q7: 如何转移域名？

**A:** 
- 可以在域名服务商之间转移
- 需要获取转移码（Authorization Code）
- 转移过程通常需要 5-7 天

```
http://localhost:3000
```

### 方式2: IP地址访问
```
http://127.0.0.1:3000
```

### 方式3: 如果配置了0.0.0.0


## ❓ 如果仍然无法访问

### 检查1: 查看终端输出
开发服务器终端应该显示类似：
```
▲ Next.js 16.0.5
- Local:        http://localhost:3000
- ready started server on 0.0.0.0:3000
```

### 检查2: 常见问题

**问题**: 浏览器显示"无法访问此网站"
- **解决**: 确认服务器正在运行，检查终端

**问题**: 显示"ERR_CONNECTION_REFUSED"
- **解决**: 服务器可能未启动，运行 `npm run dev`

**问题**: 显示"ERR_CONNECTION_TIMED_OUT"
- **解决**: 检查防火墙设置

**问题**: 页面空白
- **解决**: 等待几秒让页面加载，检查浏览器控制台（F12）

### 检查3: 浏览器控制台
1. 按 `F12` 打开开发者工具
2. 查看"Console"标签页
3. 查看是否有错误信息

### 检查4: 网络标签
1. 按 `F12` 打开开发者工具
2. 查看"Network"标签页
3. 刷新页面，查看请求是否成功

## 🔄 重启服务器

如果无法访问，尝试重启：

1. **停止服务器**: 在运行 `npm run dev` 的终端按 `Ctrl + C`
2. **重新启动**:
   ```bash
   npm run dev
   ```
3. **等待启动完成**: 看到 "ready started server" 消息
4. **再次访问**: http://localhost:3000

## 📱 移动设备访问

如果想在手机上访问（同一网络）：

1. 找到你的电脑IP地址:
   ```bash
   ipconfig
   ```
   查找 "IPv4 地址"，例如: 192.168.1.100

2. 在手机浏览器输入:
   ```
   http://192.168.1.100:3000
   ```
